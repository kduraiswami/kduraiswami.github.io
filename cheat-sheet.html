String method
.squeeze
Returns a new string where runs of the same character that occur in this set are replaced by a single character. If no arguments are given, all runs of identical characters are replaced by a single character.

"yellow moon and sun".squeeze
#=> "yelow mon and sun"
Array method
.pop
Removes the last element from self and returns it, or nil if the array is empty. If a number n is given, returns an array of the last n elements.

a = [ "a", "b", "c", "d" ]
a.pop #=> "d"
a.pop(2) #=> ["d", "c"]
a #=> ["a"]
Array method
.slice
Returns the element at an index, or returns a subarray starting at the start index and continuing for length elements, or returns a subarray specified by range of indices.

a = [ "a", "b", "c", "d", "e" ]
a[2] + a[0] + a[1] #=> "cab"
a[1, 2] #=> [ "b", "c" ]
a[-3, 3] #=> [ "c", "d", "e" ]
Enumerable method
.find_all
Returns an array containing all elements of enum for which the given block returns a true value. If no block is given, an Enumerator is returned instead.

(1..10).find_all { |i| i % 3 == 0 }
#=> [3, 6, 9]

[1,2,3,4,5].find_all { |num| num.even? }
#=> [2, 4]
Enumerable method
.map
Returns a new array with the results of running block once for every element in enum. If no block is given, an enumerator is returned instead.

(1..4).map { |i| i*i }
#=> [1, 4, 9, 16]

array = ["a", "b", "c", "d"]
array.map {|elem| elem + "1"}
#=> ["a1", "b1", "c1", "d1"]

JavaScript functions

Array function
.push()
The push() method adds one or more elements to the end of an array and returns the new length property of the object upon which the method was called.

var sports = ['apple', 'orange'];
sports.push('grape', 'blueberry');
#=> ["apple", "orange", "grape", "blueberry"]
Array function
.toString()
The toString() method returns a string representing the specified array and its elements. For Array objects, the toString method joins the array and returns one string containing each array element separated by commas.

var array = ["me", "you", "he", "she"]
array.toString()
#=> "me,you,he,she"
Math function
.floor()
The Math.floor() function returns the largest integer less than or equal to a given number. Because floor() is a static method of Math, you always use it as Math.floor(), rather than as a method of a Math object you created (Math is not a constructor).

var int = 45.89
Math.floor(int)
#=> 45
Math function
.max()
The Math.max() function returns the largest of zero or more numbers. max() is also a static method of Math, you always use it as Math.max(), rather than as a method of a Math object you created (Math is not a constructor).

c
Math.max(a, b)
#=> 50
String function
.toLowerCase()
The toLowerCase() method returns the called string value converted to lowercase. toLowerCase() does not affect the value of the string variable str itself.

var str = "ALPHABET"
str.toLowerCase()
#=> "alphabet"
String function
.includes()
The includes() method determines whether one string may be found within another string, returning true or false as appropriate.

var str = 'To be, or not to be, that is the question.';
str.includes('To be')
#=> true
